# The Architect — Chidi Michael Okobia's Portfolio

> *Engineering systems that scale. Building products that matter.*

A jaw-dropping, interactive developer portfolio built with cutting-edge web technologies. This isn't just a portfolio — it's a technical showcase that proves every skill it describes.

## Tech Stack

- **Framework:** Next.js 14+ (App Router, Server Components)
- **Language:** TypeScript
- **Animations:** Framer Motion
- **3D Engine:** Three.js (React Three Fiber)
- **Styling:** Tailwind CSS + CSS Custom Properties
- **Physics:** Matter.js
- **Audio:** Tone.js
- **AI:** Claude API
- **Deployment:** Vercel

## Features

- **4 Switchable Themes** — Dark Cinematic, Clean Minimal, Vibrant Bold, Glassmorphism
- **3D Interactive Hero** — Unique scene generated for every visitor
- **Interactive Terminal** — Type commands to explore the portfolio
- **Voice Navigation** — Speak to navigate the site
- **Live Code Playground** — Run real API code in the browser
- **AI Chatbot** — Ask questions about my experience
- **3D Skills Solar System** — Interactive tech stack visualization
- **Scroll-Reactive Music** — Ambient soundtrack that responds to scrolling
- **WebGL Liquid Effects** — GLSL shader distortions
- **Physics Playground** — Drag, throw, and bounce project cards
- **Collaborative 3D Guestbook** — Leave notes on a 3D wall
- **Easter Eggs** — Matrix rain, Konami code, and more

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure

```
src/
├── app/           # Next.js App Router pages & API routes
├── components/    # React components (sections, layout, effects, 3D)
├── context/       # Theme & global state
├── data/          # Static data (projects, skills, experience)
├── hooks/         # Custom React hooks
└── lib/           # Utilities, theme config, shaders
```
